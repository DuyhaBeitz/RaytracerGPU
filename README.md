# C++ & raylib raytracer
Uses regular shader
## features
- Materials: lambertian, metal, dielectric, diffuse light
- Materials can be textured, have emission, fuzz, refraction
- Lowpoly models (no texturing)
- Equirectangular sky map
- different scenes
- camera focusing
